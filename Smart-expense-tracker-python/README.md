# 💡 Smart Expense Tracker with Insights

A beginner-friendly Python-based web application that helps users record, categorize, and analyze their daily expenses.  
The application provides meaningful insights and visualizations to understand spending patterns and improve budgeting habits.

---


## 🔗Streamlit App link
https://smart-expense-tracker-vikasni-06.streamlit.app/


---

## 📌 Problem Statement
Many individuals struggle to track daily expenses and understand where their money goes.  
This project aims to simplify expense tracking and provide clear monthly insights using data analysis and visualization.

---

## 🎯 Objectives
- Record daily expenses with complete details
- Categorize spending (Food, Travel, Bills, etc.)
- Store data securely in CSV format
- Generate monthly summaries and insights
- Visualize category-wise spending
- Identify high-spending areas and suggest improvements

---

## 🛠 Tech Stack
- **Language:** Python 3
- **Framework:** Streamlit
- **Libraries:** pandas, matplotlib
- **Data Storage:** CSV
- **IDE:** Visual Studio Code
- **Version Control:** Git & GitHub
