# 🛒 Online Retail Sales Analysis (SQL Project)

---

## 📌 Project Overview
This project is a relational database system designed for an online retail store.  
It focuses on storing and analyzing sales data to generate useful business insights such as product performance, customer spending, and revenue trends.

The project is implemented using MySQL and demonstrates real-world SQL analysis techniques.

---

## 🎯 Objectives

- Design a structured relational database for retail data
- Store customers, products, orders, and order items
- Perform SQL-based data analysis
- Generate meaningful business insights
- Practice joins, aggregations, and filtering techniques

---

## 🛠️ Technologies Used

- MySQL
- SQL (DDL, DML, Joins, Aggregations)
- Relational Database Design Concepts

---

## 🗂️ Database Structure

The database consists of the following tables:

### 👤 Customers
Stores customer details such as ID, name, and city.

### 📦 Products
Stores product information including name, category, and price.

### 🧾 Orders
Stores order details linked to customers with order date.

### 🛍️ Order Items
Stores product-level details for each order including quantity purchased.

---

## 📥 Sample Data

The database includes sample data for:
- 5 customers from different cities
- Multiple products across categories like Electronics, Accessories, and Furniture
- Several customer orders with different dates
- Order items representing product purchases

---

## 📊 SQL Analysis Performed

The following business insights were generated using SQL queries:

### 🏆 Top Selling Products
Identifies products with the highest quantity sold.

### 💰 Most Valuable Customers
Finds customers who have spent the most money.

### 📈 Monthly Revenue Analysis
Calculates total revenue generated per month and year.

### 🏷️ Category-wise Sales Analysis
Analyzes revenue contribution of each product category.

### 🚫 Inactive Customers
Identifies customers who have not placed any recent orders.
